# TailAdmin Template Reference

This folder contains a snapshot of the original TailAdmin Laravel template design for reference.

Use this when you need to:
- Restore original component markup
- Compare against current implementation
- Reference layout structure, classes, or patterns

Do not edit these files. They are preserved as the baseline before Forever-Loved customizations.

## Contents

- `resources/views/` - Blade templates (layouts, pages, components)
- `resources/css/` - Stylesheets
- `resources/js/` - JavaScript files
